# CVStudio

Build professional resumes in minutes.

CVStudio is a modern resume builder designed to help professionals create visually appealing, ATS-friendly resumes with real-time editing, advanced analytics, and premium templates.

---
## 🌐 Live Demo

https://rafacodehub.github.io/rafaelamorim-portfolio/

### 🎨 Professional Resume Builder

Create resumes through an intuitive visual editor with live preview.

### 📄 Premium Templates

Choose from multiple professionally designed templates:

* Minimal Clean
* Corporate Word
* Corporate Blue
* Creative
* Executive Black
* Gold Premium

Each template has its own layout and visual identity.

### 📊 ATS Score Analysis

Analyze your resume and receive a score from 0–100 based on:

* Professional Summary
* Work Experience
* Education
* Skills
* Languages
* Projects
* ATS Keywords

### 🌙 Dark Mode

Fully integrated dark mode experience.

### 📥 PDF Export

Generate high-quality PDF resumes optimized for printing and sharing.

### 📚 Export History

Track previously exported resumes and download them again at any time.

## 🚀 Why CVStudio?

Most resume builders focus only on appearance.

CVStudio combines:

* Professional design
* ATS optimization
* Resume analytics
* Export management
* Premium user experience

Helping candidates increase their chances of passing Applicant Tracking Systems and reaching recruiters.
